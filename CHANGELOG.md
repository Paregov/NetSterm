## [Unreleased]

### Added

- Cross-platform support via Avalonia UI (Windows, Linux, macOS)
- Multi-tab SSH terminal sessions with xterm.js
- Session management with encrypted credential storage (AES-256-CBC)
- SFTP file browser with drag-and-drop upload
- Command snippets with folder organization
- Quick connect toolbar
- Master password protection
- Export/Import configuration as ZIP (connections, snippets, private keys)
- Dark theme (Avalonia Fluent)
- Keyboard shortcuts (F2 rename, Delete, drag-and-drop reorder)
- Serilog logging with rolling file sink
- Windows installer (Inno Setup)
